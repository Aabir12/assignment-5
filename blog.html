What is the Document Object Model (DOM)?

The DOM is a structured representation of a web page, where every element (like text, images, links) is treated as an object that JavaScript can access and manipulate.

How do you select an element from the DOM?

For ID: document.getElementById() ,id name in first bracket
For class: document.getElementByClassName() ,class name in first bracket
etc
